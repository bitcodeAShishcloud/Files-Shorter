Guide & Security Details

User Guide

Learn how to use our secure file upload system efficiently and safely

Login & Authentication

Secure authentication using your Google account. No additional passwords required.

Click Sign In

Click the "Sign in with Google" button at the top of the page

Select Account

Choose your Google account from the list or enter your credentials

Grant Permissions

Allow the app to access your Google Drive for file uploads

Verification Complete

You'll see "Logged in!" message when authentication is successful

Google Drive Configuration

Set up your Google Drive folder to organize uploaded files automatically.

Create Folder

Create a folder in your Google Drive where files will be stored

Copy Folder ID

Open the folder and copy its ID from the URL or share link

Open Settings

Click the ⚙️ icon in the top-right corner

Save Configuration

Paste the folder ID/URL and click "Save Settings"

Name & Subject

Organize your files with your name and subject categories.

Enter Your Name

Type your name in the "Your Name" field (required)

Select Subject

Choose from predefined subjects or create a custom one

Custom Subject

Select "Other" to create a new subject category (auto-saved)

File Organization

Files will be saved as: YourName_FileName in Subject folder

Upload System

Fast, secure, and reliable file upload with progress tracking.

Select Files

Click "Choose File" and select one or multiple files

File Type

Select the file type from dropdown (PDF, PPT, Image, etc.)

Upload

Click "Upload to Drive" button to start the upload process

Confirmation

View upload summary with success/failure status for each file

Security & Privacy

Your data security is our top priority. Here's how we protect your information

Client-Side Only

All processing happens in your browser. Files never touch our servers.

Direct Upload

Files go directly from your device to your Google Drive securely.

No Data Storage

We don't store, log, or track any of your files or personal data.

OAuth 2.0

Industry-standard authentication. We never see your password.

Minimal Permissions

Only requests access to files created by this app, not your entire Drive.

Open Source

All code is public and auditable on GitHub for transparency.

Important Security Note

This application uses Google's official APIs with OAuth 2.0 authentication. Your files are uploaded directly to your personal Google Drive account. We do not have access to view, modify, or share your files. The application only creates folders and uploads files as instructed by you. You can revoke access at any time from your Google Account settings.
