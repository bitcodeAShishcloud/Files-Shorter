# 📄 Secure File Upload - Single-File Version

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen)](https://bitcodeashishcloud.github.io/Files-Shorter/one-file/)
[![Lightweight](https://img.shields.io/badge/Size-Lightweight-green)](https://github.com/bitcodeAShishcloud/Files-Shorter/tree/main/one-file)
[![Google Drive API](https://img.shields.io/badge/Google-Drive-red)](https://developers.google.com/drive)

A **lightweight, fast, and simple** file upload tool for uploading **one file at a time** directly to your Google Drive.

## 🌟 Features

✅ **Single-File Upload** - Simple, one-file-at-a-time interface  
✅ **Lightweight** - Faster loading, minimal code  
✅ **Subject Organization** - Auto-sorts files into subject folders  
✅ **100% Secure** - Direct browser-to-Drive upload  
✅ **No Backend** - Pure HTML/CSS/JavaScript  
✅ **Mobile Optimized** - Perfect for phone uploads  

## 🚀 Live Demo

**🔗 Try it now:** https://bitcodeashishcloud.github.io/Files-Shorter/one-file/

## 📋 How It Works

1. Click "Sign in with Google"
2. Select your subject (or create new)
3. Choose ONE file
4. Click "Upload"
5. Done! File is in your Drive

## 🎯 When to Use This Version?

**Use Single-File when:**
- ✅ You need to upload just 1-2 files
- ✅ You want the fastest, simplest interface
- ✅ You're on mobile with limited bandwidth

**Use [Multi-File Version](https://bitcodeashishcloud.github.io/Files-Shorter/) when:**
- ✅ You have many files to upload
- ✅ You want to see batch progress
- ✅ You need detailed upload summaries

## 🛠️ Quick Setup

1. Visit: https://bitcodeashishcloud.github.io/Files-Shorter/one-file/
2. Click **⚙️ Settings** (top-right)
3. Paste your Google Drive folder ID/URL
4. Click **Save**
5. Start uploading!

## 🔒 Security

🔐 No server - Files never leave your browser until they reach Google Drive  
🔐 Your Drive only - Files go to YOUR Google Drive, nowhere else  
🔐 Open source - All code is public and auditable  

## 🐛 Common Issues

| Problem | Solution |
|---------|----------|
| Can't upload | Make sure you're signed in with Google |
| "Folder not found" | Set your Drive folder ID in Settings ⚙️ |
| Button gray/disabled | Complete Google sign-in first |
